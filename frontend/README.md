# CreatorGuru Frontend

This is a Next.js frontend app for CreatorGuru.

## Run Locally

```bash
npm install
npm run dev
```

The app runs on `http://localhost:3000`.

## Purpose
- Provide a polished MVP user interface for the CreatorGuru product.
- Serve as the UI layer for backend API routes in `../backend`.
- Use mock data now, then progressively switch to live API calls.
